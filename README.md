# wrp-wrp.github.io

Personal blog based on Hugo. Live at <https://wrp-wrp.github.io/>.

## 文档导航

按使用频率排序：

| 何时看 | 文档 |
|---|---|
| 日常写一篇文章 | [`QUICKSTART.md`](QUICKSTART.md) — 4 条命令速查 |
| 站点挂了 / CI 失败 / 想改配置 | [`MAINTENANCE.md`](MAINTENANCE.md) — 维护与排错 |
| 想用更稳定的写作节奏 | [`WRITING_WORKFLOW.md`](WRITING_WORKFLOW.md) — 60 分钟单篇流程 |
| 完整命令参考 | [`BLOG_WORKFLOW.md`](BLOG_WORKFLOW.md) |
| 文章里要插图库 | [`GALLERY_WORKFLOW.md`](GALLERY_WORKFLOW.md) |

辅助脚本：[`scripts/blog-workflow.sh`](scripts/blog-workflow.sh)（`help` 查看全部命令）。
