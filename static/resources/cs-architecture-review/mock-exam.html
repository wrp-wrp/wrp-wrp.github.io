建议限时 100-120 分钟。答案折叠在每题下面，先写完过程再展开。

12 分定量分析

1. CPI、MIPS 与 Amdahl

  1. 一台 800MHz 机器执行程序，整数指令占 55%，CPI=1；访存指令占 30%，CPI=3；浮点指令占 15%，CPI=4。求有效 CPI 和 MIPS。
  2. 若把访存指令的 CPI 从 3 降到 1.5，求加速比。
参考答案

有效 CPI = `0.55×1 + 0.30×3 + 0.15×4 = 2.05`。MIPS = `800 / 2.05 = 390.24`。

改进后 CPI = `0.55×1 + 0.30×1.5 + 0.15×4 = 1.60`，加速比 = `2.05 / 1.60 = 1.28`。

10 分BTB 与分支预测

2. 分支目标缓冲

条件分支占 8%，基本 CPI 为 1。BTB 命中率 85%，BTB 不命中开销 3 拍；BTB 命中后预测精度 92%，预测错误开销 7 拍。求实际 CPI。若 BTB 命中率提高到 95%，重新计算。

参考答案

额外 CPI = `0.08 × [(1-H)×3 + H×(1-0.92)×7]`。

H=0.85：`1 + 0.08×(0.15×3 + 0.85×0.08×7) = 1.07408`。

H=0.95：`1 + 0.08×(0.05×3 + 0.95×0.08×7) = 1.05456`。

12 分写策略事务数

3. Cache 写事务

一台机器 92% 访存命中 Cache，其中 25% 是写。Cache 块 64B，一次内存事务 32B，写失效采用写分配。

  1. 写直达时，一次访存平均产生多少下级事务？
  2. 写回时，30% 被替换块是脏块，一次访存平均产生多少下级事务？
参考答案

调入或写回整块要 `64/32 = 2` 次事务。

写直达：`0.92×0.25×1 + 0.08×0.75×2 + 0.08×0.25×3 = 0.41`。

写回：命中 0，下级事务只在 miss 时发生：`0.08 × (2 + 0.30×2) = 0.208`。

16 分Cache/TLB CPI

4. 写回、脏块和 TLB

基本 CPI 为 1.6，每条指令平均访问存储器 1.25 次。Cache 块 40B，主存固定延迟 30 拍，传输速率 8B/拍；Cache miss 率 2%，25% 被替换块为脏块，写回法。

  1. 理想 TLB 下实际 CPI 是多少？
  2. 若 Cache miss 产生的地址有 0.1% 发生 TLB miss，额外 30 拍，按“并入 miss penalty”口径计算实际 CPI。
参考答案

一次块读开销 = `30 + 40/8 = 35` 拍。写回平均事务倍数 = `1 + 0.25 = 1.25`，所以 penalty = `43.75`。

理想 TLB：`CPI = 1.6 + 1.25×0.02×43.75 = 2.69375`。

实际 TLB：`penalty = 1.25×(35 + 0.001×30) = 43.7875`，CPI = `1.6 + 1.25×0.02×43.7875 = 2.6946875`。

10 分RAID 与可靠性

5. RAID10 变体

4 块磁盘组成 RAID10，两组镜像再条带。系统只有一个必要控制器，可靠度 0.98；每块磁盘可靠度 0.96。忽略其他部件。写出可靠度表达式并计算。

参考答案

一组镜像可靠度 = `1-(1-0.96)^2 = 0.9984`。两组都要正常，再乘控制器。

R = 0.98 × [1-(1-0.96)^2]^2 = 0.9769，约 97.69%
10 分互连函数

6. 互连网络函数

N=32，处理器编号 0 到 31。求 `Cube2(19)`、`PM2+3(29)`、`σ(6)`。其中 σ 为 5 位二进制循环左移一位。

参考答案

`19 = 10011B`，第 2 位取反得 `10111B = 23`。

`PM2+3(29) = 29 + 8 mod 32 = 5`。

`6 = 00110B`，循环左移得 `01100B = 12`。

10 分远程访问

7. NUMA 远程访问

处理器频率 2.5GHz，基本 CPI 为 1.2。远程访问耗时 180ns，0.4% 指令需要远程访问且处理器挂起。求实际 CPI，以及只有本地访问时比它快多少。

参考答案

2.5GHz 周期 = 0.4ns；远程访问开销 = `180/0.4 = 450` 拍。

实际 CPI = `1.2 + 0.004×450 = 3.0`。只有本地访问 CPI=1.2，所以速度比 = `3.0/1.2 = 2.5` 倍。

20 分流水线与 ILP

8. 停顿判断与多发射

  1. 有正常定向时，`LW R1,0(R2)` 后紧跟 `ADD R3,R1,R4`，停几拍？
  2. `ADD R1,R2,R3` 后紧跟 `SW 0(R5),R1`，在有 store-data 定向时停几拍？
  3. `SUB R4,R3,R2` 后紧跟 `BNZ R4,L`，按本课常见口径停几拍？
  4. 3 段标量流水线执行 20 条指令需多少个 Δt？若 4 发射超标量且没有相关冲突，近似需多少个 Δt？
参考答案

1 拍；0 拍；1 拍。3 段标量流水线时间为 `(3+20-1)Δt = 22Δt`。4 发射时 20 条指令分 5 组，近似 `(3+5-1)Δt = 7Δt`。

12 套短卷

每套 4 题，建议 20 分钟。题目短，但覆盖做题流程。

短卷 1：单位、CPI、Amdahl

  1. 2.5GHz 的时钟周期是多少 ns？
  2. 基本 CPI=1.2，0.4% 指令远程访问，远程访问 180ns，2.5GHz。实际 CPI？
  3. 某程序 35% 时间可被加速 4 倍，总加速比？
  4. 600MHz、平均 CPI=1.5，MIPS 是多少？
答案

0.4ns；3.0；`1/(0.65+0.35/4)=1.36`；400 MIPS。

短卷 2：流水线停顿

  1. 有正常定向：`LW R2,0(R1); ADD R3,R2,R4` 停几拍？
  2. 有正常定向：`ADD R2,R3,R4; SUB R5,R2,R6` 停几拍？
  3. 有 store-data 定向：`ADD R2,R3,R4; SW 0(R1),R2` 停几拍？
  4. 按本课口径：`SUB R7,R1,R2; BNZ R7,L` 停几拍？
答案

1；0；0；1。

短卷 3：BTB 和循环预测

  1. 分支占 12%，BTB 命中 90%，不命中 2 拍，命中后预测错率 6%，错预测 6 拍。额外 CPI？
  2. 一位预测器初始预测不跳，一个循环 20 次，前 19 次跳，最后 1 次不跳。本轮预测准确率？
  3. BTB miss 和预测错误能不能简单相加？
  4. BTB 主要保存什么？
答案

`0.12×(0.1×2+0.9×0.06×6)=0.06288`；18/20=90%；不能，通常 BTB 命中后才谈预测路径错误；分支 PC 标识和目标地址。

短卷 4：两级 Cache

  1. L1 命中时间 1，L1 miss 率 5%，L2 命中时间 12，L2 局部 miss 率 20%，内存开销 80。AMAT？
  2. 若基本 CPI 已含 L1 命中，平均每条指令访存 1.3 次，存储停顿/指令是多少？
  3. L1 miss 40 次，L2 miss 10 次，总访存 2000 次。L2 局部 miss 率？
  4. L2 全局 miss 率？
答案

`1+0.05×(12+0.2×80)=2.4`；`(2.4-1)×1.3=1.82`；25%；0.5%。

短卷 5：写策略事务

  1. Cache 块 64B，事务 16B，调入整块要几次事务？
  2. 写回写命中会不会立刻写主存？
  3. 写直达写命中会产生什么事务？
  4. 写回、miss 率 6%、脏块 25%，块 64B、事务 16B，平均下级事务数？
答案

4 次；不会，只置 dirty；写下一级一次；`0.06×(4+0.25×4)=0.30`。

短卷 6：TLB 口径

  1. 题目说“Cache miss 产生的地址有 0.2% TLB miss”，TLB 开销通常并到哪里？
  2. 题目说“CPU 产生指令地址或数据地址有 0.2% TLB miss”，要不要算取指？
  3. 页表和 TLB 的关系是什么？
  4. TLB miss 一定会导致页失效吗？
答案

并到 Cache miss penalty；要；TLB 是页表项的高速缓存；不一定，可能只是页表项不在 TLB。

短卷 7：RAID 与可靠性

  1. 两个可靠度 0.98 的部件并联，可靠度？
  2. 三个必要部件可靠度分别 0.9、0.95、0.99，串联系统可靠度？
  3. RAID5 相比 RAID4 主要改进什么？
  4. RAID10 与 RAID01 哪个通常更可靠？
答案

`1-0.02²=0.9996`；`0.9×0.95×0.99=0.84645`；校验分布到各盘，缓解专用校验盘瓶颈；RAID10。

短卷 8：互连函数

  1. N=32，`Cube0(18)`？
  2. N=32，`PM2-2(1)`？
  3. 5 位编号 `00101` 与 `10111` 在超立方体中的距离？

短卷 10：多处理器与一致性

  1. 监听协议适合大规模还是小规模共享总线？
  2. 目录协议按什么维护目录项？

短卷 11：同步

  1. LL/SC 中 LL 盯住的是寄存器还是内存地址？
  2. SC 成功通常表示什么？
  3. 旋转锁为什么支持 Cache 一致性后更快？
  4. sense-reversing 屏障为什么 0/1 两个值够用？

短卷 12：ILP 与多发射

  1. 3 段标量流水线执行 12 条指令需多少 Δt？
  2. 4 发射且可完全打包为 3 条长指令，约需多少 Δt？
  3. 超标量和 VLIW 的主要区别是谁决定并行发射？
  4. 寄存器重命名能消除 RAW 吗？

`3+12-1=14Δt`；`3+3-1=5Δt`；超标量主要硬件动态决定，VLIW 主要编译器静态打包；不能，RAW 是真相关。
